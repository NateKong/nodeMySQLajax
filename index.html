This is what happens

  1. You use node (express actually) to start app.js
  2. If it works terminal will say something like "Express server listening on port 3000 in development mode"
  3. You opened a browser and went to localhost:3000. This call went to app.js which opened index.html
  4. In the index.html the document waits until the DOM becomes safe. Then it uses ajax to post a query to MySQL
  5. app.js receives the post from index.html and send the query to MySQL
  6. app.js gets the results from MySQL and sends it to index.html.
