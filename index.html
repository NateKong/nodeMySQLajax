This is what happens

  1. You use node (express actually) to start app.js
  2. If it works, terminal will say something like "Express server listening on port 7000 in development mode"
  3. You opened a browser and went to localhost:7000. This call went to app.js which opened index.html through app.get
  4. In the index.html the document waits until the DOM becomes safe. Then it uses ajax to post app.js
  5. app.js receives the post and sends the query to MySQL
  6. app.js gets the results from MySQL parses the JSON to a HTML table and sends it to index.html.
